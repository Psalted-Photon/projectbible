## 1. PLEIADES GAZETTEER
**Website:** https://pleiades.stoa.org/  
**Coverage:** Ancient Mediterranean, Near East, and biblical lands (Britain to India, 1000 BCE - 640 CE)  
**License:** Creative Commons Attribution 3.0 (CC-BY)

### Available Datasets

#### A. Quarterly Archival Releases (RECOMMENDED)
- **GitHub:** https://github.com/isawnyu/pleiades.datasets/releases
- **Zenodo:** https://doi.org/10.5281/zenodo.1193921
- **NYU Archive:** http://hdl.handle.net/2451/34305
- **Archive.org:** https://archive.org/search?query=Pleiades+Datasets&sort=creator&and%5B%5D=subject%3A%22ancient+geography%22
- **Latest Release:** 4.1 (May 28, 2025)
- **Update Frequency:** Quarterly
- **Formats:** All formats below bundled together

#### B. JSON Export (COMPREHENSIVE - PREFERRED)
- **Download:** https://atlantides.org/downloads/pleiades/json/
- **File:** `pleiades-places-latest.json.gz`
- **Update:** Daily (week's worth kept)
- **Content:** All attributes of all published places, names, locations, and connections
- **Format:** Gzipped JSON
- **Individual Place JSON:** Add `/json` to any place URL (e.g., http://pleiades.stoa.org/places/579885/json)

#### C. CSV for GIS (GIS-Ready)
- **Download:** https://atlantides.org/downloads/pleiades/gis/
- **Update:** Daily
- **Content:** Multiple CSV files designed for QGIS
- **Note:** NOT compatible with Microsoft Excel (use LibreOffice or Apple Numbers)
- **Best for:** GIS applications when JSON not suitable

#### D. KML/KMZ Export
- **Download:** https://atlantides.org/downloads/pleiades/kml/
- **Update:** Daily (week's worth kept)
- **Software:** Google Earth, QGIS
- **Caveat:** Only includes places with published locations (no calculated representative points)
- **Individual Place KML:** Add `/kml` to any place URL

#### E. RDF/Turtle Export
- **Download:** https://atlantides.org/downloads/pleiades/rdf/
- **Update:** Weekly (Sundays)
- **Format:** TAR archive of Turtle (Terse RDF Triple Language), gzip-compressed
- **Content:** Places, errata, authors, place types, time periods
- **Individual Place RDF:** Add `/turtle` to any place URL

#### F. Legacy CSV (DEPRECATED)
- **Download:** https://atlantides.org/downloads/pleiades/dumps/
- **Files:** `pleiades-*-latest.csv.gz`
- **Status:** For backward compatibility only - DO NOT USE FOR NEW PROJECTS
- **README:** http://atlantides.org/downloads/pleiades/dumps/README.txt

#### G. Pleiades GeoJSON (Derived)
- **GitHub:** https://github.com/ryanfb/pleiades-geojson
- **Generated by:** Ryan Baumann (Duke University Libraries)
- **Format:** GeoJSON
- **Note:** Abridged dataset - not all attributes included
- **Update:** Regular (generated from CSV export)

#### H. Pleiades Plus (Experimental)
- **GitHub:** https://github.com/ryanfb/pleiades-plus
- **Content:** Machine alignment between Pleiades and GeoNames
- **Creator:** Leif Isaksen / Ryan Baumann
- **Format:** Linking URIs
- **README:** https://github.com/ryanfb/pleiades-plus/blob/master/README.md

## 2. ANCIENT WORLD MAPPING CENTER (AWMC)
**Website:** https://awmc.unc.edu/  
**Affiliation:** University of North Carolina at Chapel Hill  
**License:** ODbL-1.0 (Open Database License)

### Available Datasets

#### A. AWMC Geodata GitHub Repository (PRIMARY SOURCE)
- **GitHub:** https://github.com/AWMC/geodata
- **License:** http://opendatacommons.org/licenses/odbl/1.0/
- **Format:** GeoJSON (primary), Shapefiles (archived)
- **Base Data:** Barrington Atlas + AWMC modifications to OpenStreetMap

##### Physical Data Directory
- Coastlines
- Inland water bodies
- River lines
- Other physical features

##### Cultural Data Directory
- Aqueducts
- Roads
- Regional names
- Human-made and cultural features

##### Shapefile Archives (for ESRI software)
- **Physical Shapefiles:** `Physical Shapefiles Apr 2024.zip`
- **Cultural Shapefiles:** `Cultural Shapefiles Apr 2024.zip`
- **Note:** GeoJSON files are more up-to-date

**Direct Download URLs:**
- Main Repository: https://github.com/AWMC/geodata
- Clone: `git clone https://github.com/AWMC/geodata.git`

#### B. CAWM Map Tiles
- **Website:** http://cawm.lib.uiowa.edu/index.html
- **Host:** University of Iowa Libraries
- **Purpose:** Base layer tiles for interactive mapping
- **Source:** Built using AWMC geophysical data
- **Contact:** thestudio@uiowa.edu

## RECOMMENDED WORKFLOW

### For Biblical Maps:
1. **Ancient Places:** Start with Pleiades JSON or CSV (comprehensive)
2. **Physical Geography:** Use AWMC GeoJSON for ancient coastlines/rivers
3. **Modern Basemap:** Use Natural Earth 10m physical vectors
4. **Labels:** Combine Pleiades places with Natural Earth populated places
5. **Verification:** Cross-reference with OpenBible.info data when available

### Best Practices:
- Always check licenses before redistribution
- Keep original attribution in your projects
- Download quarterly Pleiades releases for stable snapshots
- Use Natural Earth for base layers compatible with ancient overlays
- Combine multiple sources for comprehensive coverage
