# Polished Packs

This directory contains production-ready packs that will be bundled with the polished app (port 5174).

## Usage

1. Only place FINAL, tested packs here
2. These packs will be automatically bundled during `npm run build:polished`
3. Users will receive these packs pre-installed with the app

## Important

- Keep this directory clean - only production-ready packs
- Ensure all packs have valid licenses and checksums
- Total size affects initial app download size

## Recommended Starter Packs

- kjv.sqlite (King James Version)
- web.sqlite (World English Bible)
- maps.sqlite (Biblical places and historical maps)
