# Workbench Packs

This directory contains packs for testing and development in the workbench (port 5173).

## Usage

1. Place .sqlite pack files here for testing
2. Load them in the workbench UI
3. When ready, use the "Copy to Polished" button to promote them

## Current Packs

Add your development packs here. They will NOT be included in the polished build unless explicitly copied to `packs/polished/`.
